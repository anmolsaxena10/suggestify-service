Setup

Include the Suggestify CSS in your website:

Download it from here.

<link rel="stylesheet" href="path-to-css/suggestify.css">
Axios is a required dependency. Include the following CDN: 
<script src="https://unpkg.com/axios/dist/axios.min.js"></script>

Include the Suggestify script:

Download it from here.

<script src="path-to-script/suggestify.js"></script>

Then create a new instance of Suggestify with your input field and token.

const demo = new Suggestify({
        input: document.getElementById("your-input#id"),
        form: document.getElementById("your-form#id"),
        token: '--YOUR-TOKEN-HERE--'
    });
